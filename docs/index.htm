Guide To the SU Dashboard

The definitive guide for how to navigate the system.

By: Jason Wang

This is the one stop shop tasks, events, and much more! Read this short guide to be able to use it effectively.

The site exists at http://43.142.166.77:10000/.

Before we dive into an explanation of each page, lets understand the different account types within the system. The SU Dashboard is a closed internal system. This means that it is meant for only members. Outsiders will not be able to see its content.

For members, there are 3 different account types, in descending order of permission level:

1.       Admin

2.       Power

3.       Basic

Per design, Basic accounts should be assigned to regular members, Power accounts should be assigned to group leaders and subleaders, and Admin accounts should be given to SU leaders.

Some features are only available for certain members. In the following documentation such features (or entire sections!) will be marked with For Access level: _________ to indicate this.

Now, let’s take a look at what is in each page.

Home

This is the page you see when you login. Here, you will see a quick overview of your statistics, as well as a couple of quick access buttons. All of these though, are accessible from the top bar as well!

Admin Dashboard

For Access level: admin only

Here you will find a variety of tools to manage the system. Be careful though[1], as you are free to change almost anything. These are a number of subpages for each of the tools.

Review Project Proposals

Here, administrators can review project proposals submitted by SU members. Click “View” to approve/disapprove and to make changes.

Here, submit a response that either approves the request with suggestions (you can include files as well), or deny it with a reason.

Review Absence

Members may have to be absent from certain events due to extraneous circumstances. In these situations, they will submit an absence request. On this page, administrators will be able to approve/disapprove absences. Click on view to make a verdict on an absence.

Review Questions

All members are able to submit questions. On this page, you can respond to them. To do so, press view on a question.

Respond the question on this page, and then press Respond.

User Reference

Use this page to search though all users in the system.

Groups

This is a powerful admin tool that allows you to access the group page of any of the groups in the system. You can make changes just as if you were a power user within the group. View the My Group section for more information.

Create Awards

On this page, you can grant awards to users. Press Create New to generate a new award.

Here, fill in the relevant information and press Save to create the award. If you have a digital document for the award, add it in the files section. To see what an award in the system looks like, view the Awards section.

Create Tasks

On this page, you can arbitrarily create tasks and drop them into any group. It is recommended, however, that you instead create tasks using the Groups feature of the Admin Dashboard.

My Groups

Here you will be able to access the group you are a part of. Click on the group to enter.

On this page, you will see the members of your group as well as completed and pending tasks.

For Access level: Power and Above

Note: Do not use the Add Task Repeater button, it is still in beta and has known bugs. Do not enable the repeat task feature when creating tasks either. This should be resolved in a future update.

Click the Add Tasks button to create a new task for group members.

Most of the options on this page are self-explanatory, but there are a few special features on the bottom that you are able to set:

Check the first box if the task you are creating has a set point value.

Check the second box to automatically set the task as competed. This would be done for already completed tasks that were not previously inputted into the system.

Do not check the third box. It has known errors, which will be resolved in a future update.

My Tasks

This is a faster way to access the tasks from your group. Here you can see al list of all these tasks you need to do. Click View Completed Tasks to see all of the tasks you completed.

Profile

Here users can create custom profile pages that are viewable to all members of the organization. Click Edit Page to make changes.

Awards

Here users can see award given to oneself. For here you can view the award and download it provided certificate document (if provided).

This is the page you see when you open an award.

Leaderboard

Here you can see global rankings of members of the organization based on points. You can filter by quarter or all.

My Project Proposals

Here you can manage and create project proposals. Click on Create New to do so.

Fill in the relevant information and then Save.

You can check on your proposal by clicking View. Here you will see a list of all of the information relating to the request. Check the status of our proposal by viewing the approval section at the very bottom. If your proposal has been reviewed comments will show up in the Feedback/Feedback Files sections.

Report

Here you can see a report of all the work you have done in the organization. You can filter this report by different time frames. To print the page, follow the instructions at the top of the page.

My Events

For Access level: Power and Above

Documentation is still a work in progress.

My Events

Documentation is still a work in progress.

Questions

Here you can ask questions that will be answered by admin users. Click Create New to ask a new question. Responded questions will be displayed in Answered Questions. Click on a question there to see the response.

Bug Reports

Submit bug reports here! We would greatly appreciate if everyone submits a bug report if they encounter one. Please make sure that your submission is as descriptive as possible, and try to include screenshots in the files section.

Rules

Here you will find files and important documents.
